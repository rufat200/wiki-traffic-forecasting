# Wikipedia Web Traffic Forecasting

Проект по прогнозированию посещаемости страниц Википедии с использованием градиентного бустинга (LightGBM) и продвинутого Feature Engineering.

## О проекте
Цель проекта - разработка высокоточной модели прогнозирования трафика временных рядов. Модель учитывает инерционность данных, недельную сезонность и индивидуальные характеристики страниц, достигая метрики **SMAPE ~12%**.

## Ключевые особенности
- **Продвинутый Feature Engineering:** Лаговые признаки (1, 7, 14, 21 день), скользящие окна (Mean, Std, Max), Z-score и наклон тренда.
- **Высокая производительность:** Использование модуля на **Cython** для ускорения вычислений в 100+ раз.
- **Современный стек:** Управление зависимостями через `uv`.

## Установка и запуск
Проект использует [uv](https://docs.astral.sh/uv/getting-started/installation/) для управления окружением.

### 1. Системные требования (Windows)
Для корректной работы скомпилированных модулей C++ необходимо установить:
- [Microsoft Visual C++ Redistributable (x64)](https://aka.ms/vs/17/release/vc_redist.x64.exe)

### 2. Запуск
```bash
# Клонируйте репозиторий
git clone https://github.com/rufat200/wiki-traffic-forecasting.git/wiki-traffic-forecasting

cd wiki-traffic-forecasting

# Запустите проект
uv run main.py
```
